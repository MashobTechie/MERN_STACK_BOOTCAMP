Simple Contact List Exercise

Objective: Use your knowledge of variables, functions, and arrays to create and manage a basic list of contacts using JavaScript and console.log().

1. Set Up Your Data

  • Create a new variable called contactList using let.
  • Assign an array of at least three contact objects to this variable.
  • Each contact object should have two properties: name (a string) and number (a string).

Example of one contact object:

{
  name: "John Doe",
  number: "555-1234"
}

2. Write Your Functions

Create the following functions. Use console.log() inside each function to test results:

Function 1: addContact()

  • Add a new contact to your list.
  • Parameters: contactName, contactNumber
  • Create a new object and use contactList.push()
  • console.log() the updated contactList

Function 2: displayContacts()

  • No parameters.
  • Loop through the array and log each contact’s name and number individually.

Function 3: searchContact()

  • Parameter: searchName
  • Use if/else if to check for a matching name
  • Log the result or a "Contact not found" message.

Function 4: deleteContact()

  • Parameter: index
  • Use splice() to remove a contact.
  • Log the updated contactList

3. Test Your Functions

Call each function in your JavaScript file and check the output using the browser's console.
  • addContact("Alice", "123-4567")
  • displayContacts()
  • searchContact("John Doe")
  • searchContact("Jane") (non-existing)
  • deleteContact(0) (removes first contact)

When you run your JavaScript file, you should see all of the results using console.log() in the browser’s developer console.
